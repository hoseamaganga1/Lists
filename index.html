  • Website Components

    1. HTML- HYPER TEXT MARKUP LANGUAGE
      • Responsible for writing the actual content of the page.
      • It has many elements to enable the developer to include difeerent elements. E.g lists,heading, paragraphs e.t.c
    2. CSS- CASCADING STYLE SHEETS
      • Responsible for making the page beautiful/attractive.
      • Contains many properties to achieve this. E.g color, background e.t.c
    3. JS- JAVASCRIPT
      • Responsible for functionality on the site.
      • It achieves this through methods & functions that do all sorts of things. E.g control when certain components load or react to the user input e.t.c
